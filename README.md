# health
SpringBoot+React+Mysql+Mybatis+Restful风格接口，前后端分离实现健康管理系统
## 启动步骤

后端启动（lesson）
1. 将创建数据库，执行sql文件
2. 运行后端

前端启动（code-front-end）
1. 安装nodejs
2. 修改镜像源
$ npm install -g cnpm --registry=https://registry.npm.taobao.org
3. 安装依赖包
$ cnpm install
4. 启动
$ npm start
